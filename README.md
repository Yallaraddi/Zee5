#### Run in your VPS

* Install requirements

```sh
sudo apt install python3-pip
sudo apt install ffmpeg
```

* Create config.py appropriately (Refer Configs section)

* Run the app

```sh
git clone https://github.com/Yallaraddi/Zee5
cd Zee5-Downloader
pip3 install -r requirements.txt
python3 bot.py
```
